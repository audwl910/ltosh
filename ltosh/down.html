파이썬 공식홈페이지    파이썬 공식홈페이지의 다운페이지
파이썬 설치 방법
우선 파이썬 공식 홈페이지의 다운로드 페이지에서
윈도우용 파이썬 언어 패키지를 다운로드 합니다.
다음 화면에서 Python 3.x로 시작하는 버전 중
가장 최근의 윈도우 인스톨러를 다운로드하도록 합니다.
(이 글을 작성하는 시점의 최신 버전은 3.6.5.).
인스톨러를 실행한 후에 "Install Now"를 선택하면
바로 설치가 진행됩니다.
파이썬이 어느 곳에서든지 실행될 수 있도록
"Add Python 3.6 to PATH" 옵션을 선택하도록 합시다.
설치가 완료되면 [close]를 클릭하여 종료합니다.
파이썬이 정상적으로 설치되었다면 오른쪽 그림과
같이 프로그램 메뉴에서 확인할 수 있습니다.
[시작 → 모든 프로그램(모든 앱) → Python 3.6]
파이선 추천 에디터
IDLE

파이썬 IDLE는 파이썬 프로그램 작성을 도와주는
통합 개발환경으로 파이썬 설치시 기본으로 설치되는 프로그램
비주얼 스튜디오 코드

비주얼 스튜디오 코드는 파이참과 더불어 가장 많은 사랑을
받고 있는 파이썬의 대표적인 에디터이다.
비주얼 스튜디오 코드는 공식 다운로드 사이트에서 다운로드할 수 있다.
비주얼 스튜디오 코드는 파이썬 전용 에디터가 아니다.
(파이썬 외에 여러가지 언어를 지원하는 에디터이다.)
따라서 비주얼 스튜디오 코드 설치 후 파이썬 편집을 위해
가장 먼저 해야 할 일은 파이썬 Extension을 설치하는 일이다.
파이썬 Extension은 비주얼 소스 코드 실행 후 Extension메뉴를
이용하여 설치할 수 있다.
파이참

파이썬에 어느 정도 익숙해졌다면 파이참을 사용해 보기를 적극 추천한다.
파이참은 가장 유명한 파이썬 에디터 중 하나로 코드 작성 시
자동 완성, 문법 체크 등 편리한 기능들을 많이 제공한다.
파이참은 파이썬 전용 에디터로 별도의 Extension이나
플러그인을 따로 설치할 필요가 없다.
파이썬 이용약관 개인정보 처리방침 책임의 한계와 법적 고지 학급 서비스 이용약관 파이썬 이용정책 파이썬 학술 자료
